# Travel App
 
 Aplicativo de viagem onde o usuário pode selecionar as datas de ida e volta, bem como informações do local de destino.

 O projeto é requisito para conclusão do curso "Desenvolvedor Front-End" da Udacity.

 Os requisitos exigidos no projeto são os seguintes:

 * O usuário deve poder escolher a data de início e fim da viagem, bem como o local;
 * O aplicativo deve exibir o clima do local no período selecionado;
 * O aplicativo deve exibir uma imagem do local.

 As informações de clima e do local deverão ser obtidas por meio do uso de API's externas.
